Hillsboro to Yellowstone

The purpose of this map is to serve as a travel guide for anyone intending to drive from Hillsboro, Oregon to Yellowstone National Park in Wyoming. There are three main routes that a traveller could choose from while planning a trip of this nature, each with their own unique attractions and places of interest. Given the differences between each route, a typical traveller may want to weigh the options of one route against another. One possible way to accomplish this goal would be to use an interactive web map that allows for quick reference across multiple routes. In this case, the page consists of two main components.The first component is the main map, which displays the three routes and markers for each point of interest. The second component is a navigation panel that displays all the possible points on each route and allows the user easy navigation to each marker. This would allow a user to browse road trip stops in an intuitive and easy to process manner.
